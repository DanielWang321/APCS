Java is an object-oriented programming language. A class defines the attributes and behaviors of objects, while an object is an instance of a class. Constructors initialize new objects and have the same name as the class. Instance variables store an object's state, and methods define its behavior. A for loop is useful when the number of repetitions is known, while a while loop repeats as long as its condition is true. Arrays have a fixed length and use zero-based indexes. ArrayList can grow or shrink and provides methods such as add, get, set, and remove. Inheritance allows a subclass to extend a superclass. Method overriding lets a subclass provide its own version of an inherited method.
